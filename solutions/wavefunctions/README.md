# Pseudo-state wavefunctions for $l = 0$, $1$, and $2$

This directory contains the first 10 pseudo-state wavefunction for $l = 0$, 
$1$, and $2$. These are obtained with sturmian basis with $\zeta = 1.0$. These
files are formatted in the following way. The first column is the electron 
radial grid and the remaining columns are the first, second, etc. pseudostate
wavefunctions for that $l$.
